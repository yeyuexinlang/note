# 快捷键

## 打开项目

`ctrl + K & ctrl + O`

## 打开文件

`ctrl + P` , 输入文件名

## 调用命令

**打开命令框** `ctrl + shift + P`

## 终端

**打开终端** ```ctrl + ` ```

**新建终端** ```ctrl + shift + ` ```

## 文件操作

**跳转至行** 
- `ctrl + G`, 输入行号  
- `ctrl + P`, `:` + 行号

**按单词移动** `ctrl + left|right`

**按单词删除** `ctrl + backspace`

**选中单词** 
- `ctrl + D`
- 多次点击 `ctrl + D` 选择多个相同单词

**删除行** `ctrl + shift + K` | `shift + delete`

**行中换行** `ctrl + enter`

**注释一行** `ctrl + /`

**添加一个缩进** `ctrl + ]`

**去除一个缩进** `ctrl + [`

**向上|向下移动行** `alt + up|down`
